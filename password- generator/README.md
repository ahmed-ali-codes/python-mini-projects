# Password Generator

A simple and secure command-line password generator written in Python.

## Features

- **Customizable Length**: Choose exactly how long you want your password to be.
- **Numbers & Symbols**: Option to include numbers and special characters for stronger security.
- **Randomized Security**: Uses Python's built-in `random` and `string` modules to generate strong passwords.

## Concepts Demonstrated

- String Manipulation
- The `random` and `string` modules
- Functions and modularity
- User input handling and validation

## Getting Started

### Prerequisites

You need Python 3 installed on your system. You can download it from [python.org](https://www.python.org/).

### Running the Program

1. Clone or download this repository.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the project.
4. Run the script using Python:

   ```bash
   python password_generator.py
   ```

## Usage

When you run the script, you will be prompted to:
1. Enter the desired length of your password.
2. Choose whether to include numbers (`y/n`).
3. Choose whether to include symbols (`y/n`).

The program will then generate and display your secure password instantly.
