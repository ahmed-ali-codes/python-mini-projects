# Python Quiz Game

A fun, interactive command-line quiz game built with Python. This project demonstrates core programming concepts such as dictionaries, loops, and randomizing data.

## Features
- **Multiple Questions**: A built-in set of diverse questions.
- **Randomized Order**: Questions appear in a different order each time you play to keep the challenge fresh.
- **Score Tracking**: Keeps track of your correct answers and displays your final score at the end.

## Concepts Used
- **Dictionaries**: Used to store questions as keys and the correct answers as values.
- **Loops**: Used to iterate through the questions and process user input.
- **Random Module**: Used to shuffle the question order.

## How to Play
1. Ensure you have Python 3 installed on your system.
2. Clone or download this repository.
3. Open a terminal and navigate to the project directory.
4. Run the game using the following command:
   ```bash
   python quiz.py
   ```
5. Type your answer and press Enter. To exit the game early, type `quit`.
