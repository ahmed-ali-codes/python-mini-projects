# Number Guessing Game

A simple, interactive command-line Number Guessing Game written in Python. This project demonstrates core programming fundamentals including variables, loops, conditionals, and random number generation.

## Features

- **Random Number Generation**: The game randomly selects a secret number between 1 and 100.
- **Interactive Gameplay**: Users are prompted to enter their guesses.
- **Helpful Feedback**: Provides "Too high!" or "Too low!" hints to guide the user towards the correct number.
- **Attempt Tracking**: Accurately counts and displays the total number of attempts it took to win.
- **Input Validation**: Gracefully handles non-integer inputs without crashing the program.

## Prerequisites

- Python 3.x

## How to Run

1. Clone this repository or download the source code.
2. Open your terminal or command prompt.
3. Navigate to the project directory.
4. Run the script using Python:

   ```bash
   python number_guessing_game.py
   ```

## How to Play

1. Upon starting the game, you will be informed that a secret number between 1 and 100 has been selected.
2. Enter your guess when prompted.
3. If your guess is incorrect, the game will tell you if your guess was "Too high" or "Too low".
4. Keep guessing until you find the correct number!
5. Once you guess correctly, the game will congratulate you and tell you how many attempts it took.
