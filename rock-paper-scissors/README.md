# Rock-Paper-Scissors

A classic Rock-Paper-Scissors game written in Python. Play against the computer in either a Best-of-3 or Best-of-5 mode!

## Features
- Interactive command-line interface.
- Play against a computer opponent that makes random choices.
- Keeps track of the score across rounds.
- Choose between **Best of 3** (first to 2 wins) or **Best of 5** (first to 3 wins) game modes.

## Concepts Demonstrated
- Conditionals (`if`, `elif`, `else`)
- Loops (`while`)
- User input handling and validation
- Random choices using the `random` module

## Prerequisites
- Python 3.x installed on your machine.

## How to Run
1. Clone or download this repository.
2. Open a terminal or command prompt.
3. Navigate to the directory containing the script.
4. Run the script using Python:
   ```bash
   python rock_paper_scissors.py
   ```

## Usage
1. When prompted, select the game mode by entering `1` for Best-of-3 or `2` for Best-of-5.
2. In each round, enter `rock`, `paper`, or `scissors`.
3. The computer will randomly select its choice.
4. The winner of the round is determined, and the score is updated.
5. The game continues until either you or the computer reaches the target score.
6. Type `quit` at any time to exit the game.
