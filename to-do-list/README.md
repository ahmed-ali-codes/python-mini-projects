# Python To-Do List

A simple, interactive command-line To-Do List application written in Python. 

This project demonstrates core Python concepts such as **lists**, **functions**, and **file handling**. The application allows users to manage their daily tasks, saving them to a local text file so they persist across sessions.

## Features

- **Add tasks:** Add new items to your to-do list.
- **View tasks:** See all your current tasks with enumerated indexing.
- **Delete tasks:** Remove completed or unwanted tasks by referencing their number.
- **Persistent storage:** Automatically saves and loads your list from a `tasks.txt` file.

## Prerequisites

- Python 3.x

## How to Run

1. Clone the repository or download the files.
2. Navigate to the directory containing the script.
3. Run the application from your terminal:

```bash
python3 todo_list.py
```
